Beispiele

Das Buch enthält verschiedene Beispiele. Die Links auf den Code sind:

  • Microservices Demo ist ein vollständiges Beispiel für eine Microservices-Architektur mit Java, Spring, Spring Boot, Spring Cloud, Docker und dem Netflix-Stack.
  • WAR Demo zeigt das Deployment von mehreren WARs in einen Tomcat - ein Beispiel für Nanoservices.
  • Erlang Example ist ein Beispiel für ein sehr einfaches Erlang-System. Erlang kann ebenfalls eine gute Basis für Nanoservices sein.
  • Seneca Example zeigt, wie SenecaJS für JavaScript Nanoservices unterstützt.
